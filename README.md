Projeto: Ransomware Educacional em Python
Este projeto foi desenvolvido como parte do desafio da plataforma DIO para demonstrar, de forma educacional, como funciona o processo básico de criptografia e descriptografia de arquivos utilizando a linguagem Python.

Objetivo
O objetivo do projeto é entender, na prática, como um ransomware pode criptografar arquivos e como esses arquivos podem ser descriptografados posteriormente utilizando uma chave específica.

Este projeto tem finalidade exclusivamente educacional, voltado para o aprendizado de conceitos de cibersegurança.

Estrutura do Projeto
O projeto contém dois arquivos principais:

encrypter.py
Responsável por criptografar um arquivo utilizando um algoritmo de criptografia.

  decrypter.py
  Responsável por descriptografar o arquivo previamente criptografado, restaurando seu conteúdo original.

Tecnologias Utilizadas
* Python
* Biblioteca de criptografia

Como Funciona
1. O arquivo original é selecionado.
2. O script `encrypter.py` criptografa o conteúdo do arquivo.
3. O arquivo criptografado não pode ser lido normalmente.
4. O script `decrypter.py` utiliza a chave correta para restaurar o conteúdo original.
